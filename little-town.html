Little Town: Trick or Treat

Little Town: Trick or Treat is a cozy logic-puzzle reskin of a default GameMaker Studio project, reimagined with a spooky seasonal twist.

Built in GameMaker Studio, this project focuses on EN-FR and EN-KO localization, art adaptation, and narrative transformation rather than core engine changes.

Roles
 Game Designer
Narrative Designer
Localization & QA (EN / FR / KO)
Artist
Sound Designer

Contributions
 Fully reskinned the default GameMaker project with original Halloween-themed art
Rewrote all dialogue into a logic-based puzzle structure
Translated and QAed the full game into French and Korean using industry-standard CAT tools (SDL Trados Studio, memoQ, Smartcat)
Performed all DTP and text reintegration in Adobe Photoshop
Designed and implemented new sound effects using Audacity and Adobe Firefly

Key Features
 Familiar GameMaker gameplay enhanced with a narrative logic puzzle layer
Cozy-spooky atmosphere inspired by small-town Halloween traditions
Multilingual UI carefully adjusted to avoid truncation and layout issues
Brand-new eerie music stings and ambient sounds that reinforce puzzle tension
